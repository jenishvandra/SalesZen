# 📖 Overview

SalesZen is a modern AI-inspired Sales CRM web application designed to simplify customer relationship management and sales operations. It provides an intuitive dashboard for managing leads, monitoring sales performance, tracking customer interactions, and visualizing business insights through an elegant user interface.

The project focuses on improving productivity by bringing together sales analytics, lead management, customer information, and performance tracking in one centralized platform.
